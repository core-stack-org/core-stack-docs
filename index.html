    Welcome to CoRE Stack¶

    CoRE Stack is an open geospatial platform for public data access, computation pipelines, and community-grounded planning.

    The fastest way to use this docs site is to choose the route that matches your intent, then follow the next links without needing to learn the whole stack first.

    Choose Your Route¶

    Learn the hydrological units, identifiers, and cross-links that make CoRE Stack data usable.

    Open Data Structure

    • Use PreComputed Geospatial Data

    Discover valid places, obtain API access, inspect public endpoints, and download assets through STAC and QGIS.

    Use PreComputed Data

    • Develop Computation Pipelines
