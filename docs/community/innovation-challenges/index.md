# Innovation Challenges

CoRE Stack currently supports two challenge-linked entry points for people who want to build with geospatial data, remote sensing, and public-interest technology.

## Current Calls

- [CoRE Stack Innovation Challenge](core-stack-innovation-challenge.md)
- [TESSERA Student Innovation Call](tessera-student-innovation-call.md)

## How The Two Calls Relate

TESSERA is the more model-centered student research path.

The CoRE Stack Innovation Challenge is the broader application and integration path.

Projects can span both tracks when the work fits both sets of goals.
