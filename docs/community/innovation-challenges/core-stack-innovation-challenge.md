# CoRE Stack Innovation Challenge

The CoRE Stack Innovation Challenge launched on `November 22, 2025`. As of `March 11, 2026`, it is being run as a rolling contribution program rather than a fixed-deadline contest.

The challenge invites ecologists, hydrologists, geospatial programmers, students, NGOs, and community technologists to build open-source analyses, tools, and integrations using CoRE Stack datasets and APIs.

---

## Challenge Tracks

- **Data exploration and insights**: reproducible analyses, maps, and indicators built from CoRE Stack layers
- **Tooling, APIs, and developer tools**: wrappers, libraries, utilities, and workflow helpers
- **Data stories and UX integrations**: dashboards, chat interfaces, practitioner tools, and communication-oriented products

---

## Typical Submission Package

- a public repository with source code and documentation
- at least one reproducible notebook or script
- a short demo video or report
- a concise project summary and metadata
