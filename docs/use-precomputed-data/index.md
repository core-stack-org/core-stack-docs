# Use PreComputed Geospatial Data

This section is for people who want to discover, inspect, download, and reuse CoRE Stack data before installing the backend.

!!! tip "Need an API key?"
    [Register or sign in at dashboard.core-stack.org](https://dashboard.core-stack.org/){ .md-button .md-button--primary }
    Then generate an API key before using the authenticated public-data APIs.

---

## Key Pages

1. Move to [Public APIs](public-apis.md) for API keys, Swagger, ReDoc, and the public API catalog.
2. Try the [Water Balance Analysis](public-apis.md#water-balance-analysis) or [Cropping Intensity Analysis](public-apis.md#cropping-intensity-analysis) notebooks when you want a worked API example.
3. Use [STAC Specs](stac-specs.md) when you need metadata, styles, and download structure.
4. [CoRE Stack GEE App](https://ee-corestackdev.projects.earthengine.app/view/core-stack-gee-app): browse public layers directly in Earth Engine.
5. [How Current Data Was Computed](how-current-data-was-computed.md): when you want to understand which pipeline family produced a dataset.

---

## If The Data You Need Is Missing

You should not have to guess what to do next. Move into:

- [Pipeline Development](../pipelines/index.md) if you want to install and compute locally
- [Contributing](../community/contributing.md) if you want to request or help add missing data
- [Geospatial Data Request Form](https://dashboard.core-stack.org/locationForm/)
- Check current Issues on Github or [Raise a new Issue](https://github.com/core-stack-org/core-stack-backend/issues)
