## Research Repositories Around CoRE Stack

These repositories are not just general geospatial tools. They are research or project repositories that are directly connected to problem areas CoRE Stack works on.

| Repository | What it does | Relation to CoRE Stack |
|---|---|---|
| [Dhruvi-Goyal/Plantation-Site-Suitability](https://github.com/Dhruvi-Goyal/Plantation-Site-Suitability) | Site-suitability analysis for tree plantation using climate, soil, ecology, and socio-economic factors. | Directly aligned with the plantation site-suitability workflow exposed in the backend. |
| [aatifnisar01/Ponds_and_wells_detection](https://github.com/aatifnisar01/Ponds_and_wells_detection) | Detection and processing work around ponds and wells. | Closely aligned with CoRE Stack’s pond, well, and merged SWB workflows. |
| [ramank1137/Scrubland-Field-Delineation](https://github.com/ramank1137/Scrubland-Field-Delineation) | Field delineation work around scrubland mapping. | Likely informative for LULC and scrubland-related classification work in the stack. |
